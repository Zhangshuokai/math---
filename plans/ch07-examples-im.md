### Ex01：可分离变量方程求通解

**文件名**：`Sec10_Examples.tsx`（复用已有文件，替换为规范6阶段）
**Composition ID**：`Ch07-Ex01-Separable`
**durationInFrames**：`450`（无3D场景）

**题目**：
> 求微分方程 $\dfrac{dy}{dx} = 2xy$ 的通解。

**对应节**：§7.2 可分离变量的微分方程

#### Stage 1（0–60帧）：展示题目
```
ExampleBox 内容：
  标题：例题1 · 可分离变量方程
  题目：求微分方程 dy/dx = 2xy 的通解
  右下角淡入"关键词：变量分离"
```

#### Stage 2（60–150帧）：审题分析
逐条 fade in，每条间隔 20 帧：
1. 🔍 **识别类型**：右侧 `2xy` 可写成 `x` 与 `y` 的乘积 → 可分离变量
2. 📐 **分离策略**：将含 `y` 的项移到左侧，含 `x` 的项移到右侧
3. ⚖️ **注意事项**：分离时需 `y ≠ 0`（后验证 `y=0` 是否为解）
4. 🎯 **求解目标**：两边积分得通解，含任意常数 `C`

#### Stage 3（150–330帧）：分步计算（4步 × 45帧）
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 分离变量：$\dfrac{dy}{y} = 2x\,dx$ |
| Step 2 | 195–240 | 两边积分：$\int \dfrac{1}{y}\,dy = \int 2x\,dx$ |
| Step 3 | 240–285 | 求积分：$\ln|y| = x^2 + C_1$ |
| Step 4 | 285–330 | 化简：$y = Ce^{x^2}$（其中 $C = \pm e^{C_1}$，验证 $y=0$ 也满足故 $C$ 可取0） |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：通解
  公式：y = C e^{x^2}    (C 为任意常数)
  说明：包含 y=0（C=0 时）
```

#### Stage 5：无3D场景 → 省略，总帧450

#### Stage 6（390–450帧）：总结
```
✅ 可分离变量方程的核心：将 f(y)dy = g(x)dx，再两边积分
✅ 注意 y=0 等奇点是否为奇解
✅ 本题通解：y = Ce^{x²}
```

### Ex02：一阶线性方程求特解

**文件名**：`Sec10_Examples.tsx`（第2个 Composition 或同文件切换）
**Composition ID**：`Ch07-Ex02-LinearFirst`
**durationInFrames**：`540`（含2D函数图像，用 FunctionPlot 替代3D）

> 注：本题图形为2D曲线族，Stage 5 改用 `FunctionPlot` + `CoordinateSystem` 展示解曲线族，rotateY 不适用，改为曲线逐条 fade-in 动画。

**题目**：
> 求方程 $y' + \dfrac{1}{x}y = \cos x$（$x>0$）满足初始条件 $y(\pi)=1$ 的特解。

**对应节**：§7.4 一阶线性微分方程

#### Stage 1（0–60帧）：展示题目
```
ExampleBox：
  标题：例题2 · 一阶线性方程特解
  题目：y' + (1/x)y = cos x，y(π) = 1，x > 0
  提示卡片："公式法：积分因子 μ(x) = e^{∫P(x)dx}"
```

#### Stage 2（60–150帧）：审题分析
1. 🔍 **标准形式**：$y' + P(x)y = Q(x)$，此处 $P(x)=\tfrac{1}{x}$，$Q(x)=\cos x$
2. 📐 **积分因子**：$\mu(x) = e^{\int P\,dx} = e^{\ln x} = x$
3. ⚖️ **初始条件**：解出通解后代入 $y(\pi)=1$ 确定 $C$
4. 🎯 **结果形式**：特解是一条确定曲线，不含任意常数

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 积分因子 $\mu=x$，方程两边乘 $x$：$(xy)' = x\cos x$ |
| Step 2 | 195–240 | 两边积分：$xy = \int x\cos x\,dx = x\sin x + \cos x + C$ |
| Step 3 | 240–285 | 通解：$y = \sin x + \dfrac{\cos x}{x} + \dfrac{C}{x}$ |
| Step 4 | 285–330 | 代入 $y(\pi)=1$：$0 + \tfrac{-1}{\pi} + \tfrac{C}{\pi}=1$ → $C=\pi+1$ |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：特解
  公式：y = sin x + (cos x)/x + (π+1)/x
  附注：满足 y(π) = 1
```

#### Stage 5（390–480帧）：2D 曲线族可视化
```
CoordinateSystem + FunctionPlot：
  - 背景：灰色网格，x ∈ [0.5, 2π]
  - 逐帧 fade-in 展示通解族（C = -2,-1,0,1,2 五条曲线）
  - 特解（C = π+1）用金色高亮，线宽加粗
  - 动画：曲线从左到右绘制（drawProgress 驱动）
  - 在 x=π 处标注红点，注明初始条件
```

#### Stage 6（480–540帧）：总结
```
✅ 积分因子法：μ = e^{∫P dx}，乘后变为 (μy)' = μQ
✅ 通解 = 齐次通解 + 特解（或直接积分因子公式）
✅ 初始条件确定唯一特解曲线
```

### Ex03：二阶常系数齐次方程

**Composition ID**：`Ch07-Ex03-ConstCoeff`
**durationInFrames**：`540`（含3D弹簧振动可视化）

**题目**：
> 求方程 $y'' - 3y' + 2y = 0$ 的通解，并画出解曲线。

**对应节**：§7.8 二阶常系数齐次线性微分方程

#### Stage 1（0–60帧）：展示题目
```
ExampleBox：
  标题：例题3 · 二阶常系数齐次方程
  题目：y'' − 3y' + 2y = 0
  右侧小提示："特征方程法"
```

#### Stage 2（60–150帧）：审题分析
1. 🔍 **方程结构**：常系数 → 设 $y=e^{rx}$，代入得特征方程
2. 📐 **特征方程**：$r^2 - 3r + 2 = 0$，因子分解 $(r-1)(r-2)=0$
3. ⚖️ **根的类型**：两个不同实根 $r_1=1, r_2=2$，通解为两指数函数线性组合
4. 🎯 **求解目标**：$y = C_1 e^x + C_2 e^{2x}$

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 设 $y=e^{rx}$，则 $y'=re^{rx}$，$y''=r^2e^{rx}$ |
| Step 2 | 195–240 | 代入方程：$e^{rx}(r^2-3r+2)=0$，故 $r^2-3r+2=0$ |
| Step 3 | 240–285 | 解特征方程：$r_1=1$，$r_2=2$（两不同实根） |
| Step 4 | 285–330 | 通解：$y = C_1 e^x + C_2 e^{2x}$（$C_1, C_2$ 任意） |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：通解（两不同实根情形）
  公式：y = C₁eˣ + C₂e²ˣ
  补充：特征根 r₁=1, r₂=2 → 指数增长叠加
```

#### Stage 5（390–480帧）：3D 弹簧-质量类比可视化
```
CoordinateSystem3D（rotateY: 0→360）：
  - x轴：时间 t，y轴：位移 y(t)，z轴：速度 y'(t)
  - ParametricCurve3D 绘制相平面轨迹：
      t ∈ [0, 3]，x(t)=t，y(t)=e^t+e^{2t}，z(t)=e^t+2e^{2t}
  - 曲线颜色：金色渐变
  - 旋转驱动：rotateY = interpolate(frame, [390,480], [0,360], clamp)
  - 标注三轴含义
```

#### Stage 6（480–540帧）：总结
```
✅ 特征方程将微分方程转化为代数方程
✅ 两不同实根 r₁≠r₂ → 通解 C₁e^{r₁x} + C₂e^{r₂x}
✅ 重根/复根情形见下一节
```

## 智能制造（IM）应用案例

### IM01：RC电路充放电模型

**文件名建议**：`Im01_RCCircuit.tsx`
**Composition ID**：`Ch07-IM01-RCCircuit`
**durationInFrames**：`540`

**工程背景**：
工业 PLC 控制系统中，RC滤波电路广泛用于消除传感器信号噪声。理解电容充放电过程有助于设计合适的时间常数 $\tau = RC$，避免响应过慢或震荡。

**数学模型**：
$$RC\,\frac{dU_C}{dt} + U_C = E \quad \Rightarrow \quad U_C(t) = E\left(1 - e^{-t/(RC)}\right)$$

其中 $E$ 为电源电压，$R$ 为电阻，$C$ 为电容，$U_C(0)=0$。

#### Stage 1（0–60帧）：题目展示
```
ExampleBox：
  标题：IM案例1 · RC滤波电路充电分析
  工程背景图：电路示意（ASCII或SVG简图）
  题目：已知 RC = 0.1s，E = 5V，求 U_C(t) 及达到 4V 所需时间
```

#### Stage 2（60–150帧）：审题分析
1. 🏭 **工程意义**：$\tau = RC = 0.1$ s 是时间常数，决定响应速度
2. 📐 **方程类型**：一阶线性常系数方程，可分离变量求解
3. ⚖️ **初始条件**：$U_C(0) = 0$（电容初始无电荷）
4. 🎯 **工程目标**：$U_C = 4V$ 时，$t = ?$

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 分离变量：$\dfrac{dU_C}{E-U_C} = \dfrac{1}{RC}dt$ |
| Step 2 | 195–240 | 积分：$-\ln(E-U_C) = \dfrac{t}{RC} + C_1$ |
| Step 3 | 240–285 | 代入初值：$U_C(t) = E(1-e^{-t/\tau})$，$\tau=0.1$ |
| Step 4 | 285–330 | 求时间：$4=5(1-e^{-t/0.1})$ → $t = -0.1\ln 0.2 \approx 0.161$ s |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：充电规律
  公式：U_C(t) = 5(1 − e^{−10t}) V
  工程结论：约 0.161 s 后电压达到 4V（80%充电）
  时间常数：τ = 0.1s，经过 5τ = 0.5s 基本充满
```

#### Stage 5（390–480帧）：3D 可视化
```
CoordinateSystem3D（rotateY: -30→30 小幅摇摆）：
  - x轴：时间 t (0→0.5s)
  - y轴：电压 U_C (0→5V)
  - z轴：电流 i(t) = (E/R)e^{-t/τ}（衰减曲线）
  - ParametricCurve3D 绘制 (t, U_C, i) 三维曲线
  - 在 t=0.161 处用垂线高亮标注"4V"时刻
  - AnimatedVector3D 标注斜率（充电速率）
```

#### Stage 6（480–540帧）：总结
```
✅ RC电路 = 一阶线性微分方程的经典物理模型
✅ 时间常数 τ=RC 控制响应速度：τ 越小响应越快
✅ 工程设计：根据允许延迟反推 RC 参数
```

### IM02：弹簧-质量-阻尼振动系统

**文件名建议**：`Im02_SpringDamper.tsx`
**Composition ID**：`Ch07-IM02-SpringDamper`
**durationInFrames**：`540`

**工程背景**：
CNC 机床主轴、机器人关节均存在弹性-阻尼特性。工程师需通过二阶微分方程判断系统是过阻尼、欠阻尼还是临界阻尼，以选择合适的减振材料和控制参数。

**数学模型**：
$$m\ddot{x} + c\dot{x} + kx = 0 \quad \Rightarrow \quad \ddot{x} + 2\zeta\omega_n\dot{x} + \omega_n^2 x = 0$$

其中 $\omega_n = \sqrt{k/m}$ 为固有频率，$\zeta = c/(2\sqrt{km})$ 为阻尼比。

#### Stage 1（0–60帧）：题目展示
```
ExampleBox：
  标题：IM案例2 · 机床主轴振动分析
  背景：m=1kg, c=3N·s/m, k=2N/m
  题目：求振动方程 ẍ + 3ẋ + 2x = 0 的通解，判断阻尼类型
```

#### Stage 2（60–150帧）：审题分析
1. 🏭 **物理对应**：系数 3 对应阻尼力，系数 2 对应弹性恢复力
2. 📐 **特征方程**：$r^2 + 3r + 2 = 0$，判别式 $\Delta = 9-8 = 1 > 0$
3. ⚖️ **阻尼判断**：$\Delta > 0$ → 过阻尼（两不同负实根），无振荡
4. 🎯 **工程意义**：过阻尼系统不振荡，适合精密定位但响应较慢

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 特征方程：$r^2+3r+2=(r+1)(r+2)=0$ |
| Step 2 | 195–240 | 特征根：$r_1=-1$，$r_2=-2$ |
| Step 3 | 240–285 | 通解：$x(t) = C_1 e^{-t} + C_2 e^{-2t}$ |
| Step 4 | 285–330 | 阻尼比：$\zeta = c/(2\sqrt{km}) = 3/(2\sqrt{2}) \approx 1.06 > 1$（过阻尼✓） |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：过阻尼振动
  公式：x(t) = C₁e^{-t} + C₂e^{-2t}
  工程结论：ζ ≈ 1.06 > 1，系统过阻尼，位移单调衰减至零
```

#### Stage 5（390–480帧）：3D 相平面可视化
```
CoordinateSystem3D（rotateY: 0→360）：
  - x轴：时间 t
  - y轴：位移 x(t)（取C₁=C₂=0.5）
  - z轴：速度 ẋ(t)
  - ParametricCurve3D 绘制相平面螺旋线（实际为单调衰减轨迹）
  - 对比：同一坐标系中叠加欠阻尼轨迹（虚线，颜色不同）
  - 原点标注"平衡位置"
```

#### Stage 6（480–540帧）：总结
```
✅ 阻尼比 ζ 决定振动特性：ζ>1过阻尼，ζ=1临界，ζ<1欠阻尼
✅ 工程选择：精密定位选接近临界阻尼，快速响应选欠阻尼
✅ 特征根实部为负 → 系统稳定（振动最终消失）
```

### IM03：工厂储液罐液位控制

**文件名建议**：`Im03_LiquidLevel.tsx`
**Composition ID**：`Ch07-IM03-LiquidLevel`
**durationInFrames**：`540`

**工程背景**：
化工厂、食品加工厂的储液罐需要精确控制液位。液体通过阀门流出的流量正比于液位高度，而进液泵以恒定流量注入，建立微分方程模型是实现液位自动控制的基础。

**数学模型**：
$$A\,\frac{dh}{dt} = Q_{in} - a\sqrt{h} \quad \text{（托里拆利定律）}$$

线性化近似（在工作点 $h_0$ 附近）：
$$\frac{dh}{dt} + \frac{a}{2A\sqrt{h_0}}h = \frac{Q_{in}}{A}$$

#### Stage 1（0–60帧）：题目展示
```
ExampleBox：
  标题：IM案例3 · 储液罐液位动态模型
  参数：A=2m²，Qin=0.5m³/s，a=0.2（流量系数）
  题目：设工作点 h₀=6.25m，求线性化方程的稳态液位及时间常数
```

#### Stage 2（60–150帧）：审题分析
1. 🏭 **质量守恒**：$A \cdot dh/dt$ = 进液量 − 出液量
2. 📐 **线性化**：在 $h_0$ 附近令 $h=h_0+\delta h$，忽略高阶项
3. ⚖️ **稳态条件**：$dh/dt=0$ 时 $Q_{in} = a\sqrt{h_{ss}}$，解出稳态液位
4. 🎯 **时间常数**：$\tau = 2A\sqrt{h_0}/a$，反映液位调节速度

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 稳态液位：$0.5 = 0.2\sqrt{h_{ss}}$ → $h_{ss} = 6.25$ m（与工作点一致） |
| Step 2 | 195–240 | 线性化系数：$k = a/(2A\sqrt{h_0}) = 0.2/(2\times2\times2.5) = 0.02$ s⁻¹ |
| Step 3 | 240–285 | 线性方程：$\dot{\delta h} + 0.02\,\delta h = 0$（扰动方程）→ $\delta h = \delta h_0 e^{-0.02t}$ |
| Step 4 | 285–330 | 时间常数：$\tau = 1/0.02 = 50$ s；干扰后约 $5\tau=250$ s 液位恢复 |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：液位响应规律
  公式：δh(t) = δh₀ · e^{−t/50}
  工程结论：时间常数 τ=50s，液位扰动在约 250s 内消散 95%
```

#### Stage 5（390–480帧）：3D 可视化
```
CoordinateSystem3D（rotateY: 0→180）：
  - 主体：ParametricCurve3D 绘制液位响应 h(t)
  - x轴：时间（0→300s）
  - y轴：液位 h（5→7m 范围）
  - z轴：进出流量差 Q_in − Q_out
  - 在 t=0 施加扰动 δh₀=1m，展示衰减曲线
  - 水平虚线标注稳态 h₀=6.25m
  - SurfaceMesh3D 可选：绘制液位-流量关系曲面
  - 旋转驱动：rotateY = interpolate(frame,[390,480],[0,180],clamp)
```

#### Stage 6（480–540帧）：总结
```
✅ 液位系统 = 一阶线性微分方程，时间常数 τ 决定调节速度
✅ 线性化方法：复杂非线性系统在工作点附近的标准分析工具
✅ 工程应用：PID控制器参数整定需要此时间常数信息
```
