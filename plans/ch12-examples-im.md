### Ex01：p 级数收敛性判断

**Composition ID**：`Ch12-Ex01-PSeries`
**durationInFrames**：`450`（无3D，用2D部分和动画）

**题目**：
> 讨论 $p$ 级数 $\displaystyle\sum_{n=1}^{\infty}\dfrac{1}{n^p}$ 的敛散性（$p>0$）。

**对应节**：§12.2 常数项级数的审敛法

#### Stage 1（0–60帧）：展示题目
```
ExampleBox：
  标题：例题1 · p级数的收敛性
  级数：Σ 1/nᵖ（p>0）
  特例提示：
    p=1：调和级数（发散），p=2：巴塞尔级数=π²/6（收敛）
```

#### Stage 2（60–150帧）：审题分析
1. 🔍 **特殊情况直觉**：
   - $p=1$：$1+\tfrac{1}{2}+\tfrac{1}{3}+\cdots$ → 无界（调和级数）
   - $p=2$：$1+\tfrac{1}{4}+\tfrac{1}{9}+\cdots = \pi^2/6\approx1.645$（收敛）
2. 📐 **积分判别法**：$\sum_{n=1}^\infty f(n)$ 与 $\int_1^\infty f(x)dx$ 同敛散（$f$ 单调递减正函数）
3. ⚖️ **分界点**：$p=1$ 是收敛和发散的临界值
4. 🎯 **结论**：$p>1$ 收敛，$p\leq1$ 发散

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 对应积分：$\int_1^\infty \dfrac{1}{x^p}\,dx = \left[\dfrac{x^{1-p}}{1-p}\right]_1^\infty$（$p\neq1$） |
| Step 2 | 195–240 | 当 $p>1$：$x^{1-p}\to0$（$x\to\infty$），积分 $=\dfrac{1}{p-1}<\infty$（收敛） |
| Step 3 | 240–285 | 当 $p<1$：$x^{1-p}\to\infty$，积分发散；当 $p=1$：$\int_1^\infty\tfrac{1}{x}dx=\ln x|_1^\infty=+\infty$（发散） |
| Step 4 | 285–330 | 由积分判别法：$p>1$时 $\Sigma\tfrac{1}{n^p}$ 收敛，$0<p\leq1$时发散 |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：p级数收敛定理
  结论：p > 1 → 收敛；0 < p ≤ 1 → 发散
  特殊值：p=2 → π²/6；p=4 → π⁴/90
  记忆口诀："p大于1才收敛"
```

#### Stage 5（390–450帧）：2D 可视化（替代3D）
```
FunctionPlot + CoordinateSystem（2D）：
  - 绘制三条部分和曲线 S_N 随 N 增大的变化：
      p=0.5（发散，持续上升），p=1（调和，对数上升），p=2（收敛于 π²/6）
  - x轴：N（项数），y轴：S_N（部分和）
  - p=2 曲线渐近线 π²/6 ≈ 1.645 用虚线标注
  - 三条曲线颜色区分：红（发散）/橙（临界）/绿（收敛）
  - 动画：曲线从 N=1 到 N=50 逐帧绘制
```

#### Stage 6（390–450帧，合并）：总结
```
✅ 积分判别法：将级数收敛性转化为广义积分收敛性
✅ p级数是标准参照：与其比较可判断其他级数（比较判别法）
✅ p>1 收敛 = 超过一次幂的倒数级数才能"收敛到有限值"
```

### Ex02：泰勒展开及余项估计

**Composition ID**：`Ch12-Ex02-Taylor`
**durationInFrames**：`540`

**题目**：
> 将 $f(x)=e^x$ 在 $x=0$ 处展开为幂级数（麦克劳林级数），并估计 $e$ 的近似值（取前6项，误差 $<0.001$）。

**对应节**：§12.4 函数的幂级数展开式

#### Stage 1（0–60帧）：展示题目
```
ExampleBox：
  标题：例题2 · e^x 的泰勒展开
  函数：f(x) = eˣ，展开点 x=0
  任务1：写出麦克劳林级数
  任务2：计算 e = e¹ 的6项近似值及误差估计
```

#### Stage 2（60–150帧）：审题分析
1. 🔍 **核心思想**：用无穷多项多项式 $\sum a_n x^n$ 表示 $e^x$，$a_n = f^{(n)}(0)/n!$
2. 📐 **导数规律**：$f^{(n)}(x)=e^x$，故 $f^{(n)}(0)=1$，系数 $a_n=1/n!$
3. ⚖️ **收敛半径**：$e^x$ 的泰勒级数对全实数轴收敛（$R=\infty$）
4. 🎯 **余项估计**：拉格朗日余项 $R_n = \dfrac{e^\xi}{(n+1)!}x^{n+1}$，$\xi\in(0,x)$

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 公式：$e^x = \displaystyle\sum_{n=0}^\infty\dfrac{x^n}{n!} = 1+x+\dfrac{x^2}{2!}+\dfrac{x^3}{3!}+\cdots$ |
| Step 2 | 195–240 | 代入 $x=1$：$e\approx 1+1+\tfrac{1}{2}+\tfrac{1}{6}+\tfrac{1}{24}+\tfrac{1}{120}+\tfrac{1}{720}=\dfrac{1957}{720}\approx2.71806$ |
| Step 3 | 240–285 | 误差估计：$|R_6|=\dfrac{e^\xi}{7!}\leq\dfrac{3}{5040}\approx0.000595<0.001$ ✓（用 $e<3$） |
| Step 4 | 285–330 | 精确值 $e\approx2.71828$，误差 $|2.71828-2.71806|=0.00022 < 0.001$ ✓ |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：eˣ 的麦克劳林展开
  公式：eˣ = Σ xⁿ/n! = 1 + x + x²/2! + x³/3! + ···（收敛域：−∞ < x < +∞）
  近似：e ≈ 1957/720 ≈ 2.71806（前7项，误差 < 0.001）
```

#### Stage 5（390–480帧）：3D 可视化（2D函数逼近）
```
CoordinateSystem3D（rotateY: 0→360）：
  - x轴：x值（-2到2），y轴：函数值，z轴：逼近阶数 n
  - ParametricCurve3D 绘制 Tₙ(x)（泰勒多项式，n=0到6）
    作为空间中的曲线族（不同z高度）
  - 底部 z=0：精确函数 eˣ（金色基准曲线）
  - 逼近曲线逐帧从 n=0 推进到 n=6（每3帧一阶）
  - 旋转驱动：rotateY = interpolate(frame,[390,480],[0,360],clamp)
```

#### Stage 6（480–540帧）：总结
```
✅ 麦克劳林展开：eˣ = Σ xⁿ/n!（全实轴收敛）
✅ 泰勒多项式阶数越高，逼近越精确（在展开点附近）
✅ 余项估计是工程数值计算的误差控制工具
```

### Ex03：傅里叶级数展开

**Composition ID**：`Ch12-Ex03-Fourier`
**durationInFrames**：`540`

**题目**：
> 将周期为 $2\pi$ 的函数 $f(x) = x$（$-\pi < x \leq \pi$）展开为傅里叶级数。

**对应节**：§12.6 傅里叶级数

#### Stage 1（0–60帧）：展示题目
```
ExampleBox：
  标题：例题3 · 锯齿波的傅里叶展开
  函数：f(x) = x，周期延拓，周期 2π
  物理直觉：将锯齿波分解为一系列正弦波的叠加
```

#### Stage 2（60–150帧）：审题分析
1. 🔍 **几何直觉**：$f(x)=x$ 是奇函数 → 傅里叶展开只有正弦项（$a_n=0$）
2. 📐 **系数公式**：$b_n=\dfrac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(nx)\,dx$
3. ⚖️ **奇函数简化**：$b_n=\dfrac{2}{\pi}\int_0^\pi x\sin(nx)\,dx$（利用 $f(x)\sin(nx)$ 是偶函数）
4. 🎯 **分部积分**：计算 $b_n$ 需用分部积分公式

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | $a_0=0$，$a_n=0$（$f$ 为奇函数），只需算 $b_n$ |
| Step 2 | 195–240 | $b_n=\dfrac{2}{\pi}\int_0^\pi x\sin(nx)\,dx$，分部积分：$=-\dfrac{2}{\pi}\left[\dfrac{x\cos(nx)}{n}\right]_0^\pi+\dfrac{2}{n\pi}\int_0^\pi\cos(nx)\,dx$ |
| Step 3 | 240–285 | $=-\dfrac{2\cos(n\pi)}{n}+0=-\dfrac{2(-1)^n}{n}=\dfrac{2(-1)^{n+1}}{n}$ |
| Step 4 | 285–330 | 傅里叶级数：$f(x)=2\left(\sin x-\dfrac{\sin 2x}{2}+\dfrac{\sin 3x}{3}-\cdots\right)=2\displaystyle\sum_{n=1}^\infty\dfrac{(-1)^{n+1}}{n}\sin(nx)$ |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：锯齿波傅里叶级数
  公式：f(x) = 2Σ (−1)^{n+1}/n · sin(nx)，n=1,2,3,…
  展开式：= 2(sinx − sin2x/2 + sin3x/3 − ···)
  狄利克雷定理：在不连续点 x=±π 处，级数收敛到 (π+(-π))/2 = 0
```

#### Stage 5（390–480帧）：3D 可视化
```
CoordinateSystem3D（rotateY: 0→360）：
  - x轴：时间/角度（-π到π），y轴：幅值，z轴：谐波阶数 n
  - ParametricCurve3D 绘制各阶正弦谐波（n=1,2,3,4,5），不同z高度
  - z=0 底部：原始锯齿波 f(x)=x（金色）
  - 合成动画：从n=1逐步叠加，展示波形逼近锯齿波的过程
  - FourierSeries 组件：底部2D图展示频谱（各频率分量的幅度）
  - 旋转驱动：rotateY = interpolate(frame,[390,480],[0,360],clamp)
```

#### Stage 6（480–540帧）：总结
```
✅ 傅里叶级数：任意周期函数 = 正弦/余弦波的无穷叠加
✅ 奇函数 → 只有 sin 项；偶函数 → 只有 cos 项
✅ 吉布斯现象：不连续点附近出现约9%的超调（重要工程现象）
```

### IM01：PLC数字信号低通滤波

**文件名建议**：`Im01_PLCFilter.tsx`
**Composition ID**：`Ch12-IM01-PLCFilter`
**durationInFrames**：`540`

**工程背景**：
工业PLC系统中，传感器信号常受高频噪声污染。数字低通滤波器的频率响应函数可展开为泰勒级数，工程师利用级数展开分析截止频率和滤波效果，选择合适的滤波器阶数和参数。

**数学模型**：
一阶低通滤波器传递函数（$s=j\omega$）：
$$H(\omega) = \frac{1}{1+j\omega/\omega_c} = \frac{1}{1+j\Omega}$$

泰勒展开（$|\Omega|<1$）：
$$H(\omega) = 1 - j\Omega - \Omega^2 + j\Omega^3 + \Omega^4 - \cdots = \sum_{n=0}^\infty (-j\Omega)^n$$

#### Stage 1（0–60帧）：题目展示
```
ExampleBox：
  标题：IM案例1 · 传感器信号低通滤波分析
  滤波器：一阶低通，截止频率 ω_c = 100 rad/s
  信号：含噪声，主频 10 rad/s，噪声 1000 rad/s
  问题：(1) 写出传递函数级数展开；(2) 计算主频和噪声频率的衰减比
```

#### Stage 2（60–150帧）：审题分析
1. 🏭 **工程场景**：PLC采集振动传感器信号，需滤除高频电磁噪声保留低频振动信息
2. 📐 **归一化频率**：$\Omega=\omega/\omega_c$，主频 $\Omega_1=0.1$，噪声 $\Omega_2=10$
3. ⚖️ **收敛条件**：级数展开要求 $|\Omega|<1$，仅对主频有效；噪声超出收敛域，需取模
4. 🎯 **衰减系数**：$|H(\omega)|=1/\sqrt{1+\Omega^2}$

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | 主频 $\Omega_1=10/100=0.1$：$|H|=1/\sqrt{1+0.01}=1/\sqrt{1.01}\approx0.995$（几乎不衰减✓） |
| Step 2 | 195–240 | 噪声 $\Omega_2=1000/100=10$：$|H|=1/\sqrt{1+100}=1/\sqrt{101}\approx0.0995$（衰减约 90%✓） |
| Step 3 | 240–285 | 级数展开（$|\Omega_1|=0.1\ll1$）：$H\approx 1-j(0.1)-(0.1)^2=0.99-0.1j$（幅度误差$<1\%$） |
| Step 4 | 285–330 | 滤波效果：信号保留 99.5%，噪声保留 9.95% → 信噪比改善约 **20 dB** |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：滤波性能分析
  传递函数幅度：|H(ω)| = 1/√(1+Ω²)
  主频保留：99.5%（几乎不衰减）
  噪声抑制：保留9.95%（抑制90%）
  信噪比提升：约 20 dB
```

#### Stage 5（390–480帧）：3D 可视化
```
CoordinateSystem3D（rotateY: 0→360）：
  - x轴：频率 ω（对数坐标，1→10000 rad/s）
  - y轴：幅频特性 |H|（0→1）
  - z轴：泰勒级数阶数 N（展示各阶近似的Bode图）
  - ParametricCurve3D 绘制精确 |H(ω)| 曲线（底部，金色）
  - 各阶泰勒近似曲线在不同 z 高度（N=1,2,3,4）
  - 截止频率 ω_c=100 处用垂线标注（红色）
  - 旋转驱动：rotateY = interpolate(frame,[390,480],[0,360],clamp)
```

#### Stage 6（480–540帧）：总结
```
✅ 级数展开在频率远低于截止频率时近似精度很高（|Ω|<1）
✅ 滤波器设计：截止频率 ω_c 是信号带宽的分界线
✅ 工程实践：传感器信号一般选 ω_c = 10×信号最高频率
```

### IM02：数控插补的泰勒近似

**文件名建议**：`Im02_CNCInterpolation.tsx`
**Composition ID**：`Ch12-IM02-CNCInterpolation`
**durationInFrames**：`540`

**工程背景**：
数控机床加工圆弧时，插补器将圆弧路径近似为若干小直线段（线性插补）或小圆弧。实际上，当圆弧段很短时，用正弦/余弦函数的泰勒展开来简化三角函数计算，可大幅降低实时计算量，同时控制插补误差在允许范围内（通常 < 1μm）。

**数学模型**：
圆弧插补中需要计算 $\cos\Delta\theta$ 和 $\sin\Delta\theta$（$\Delta\theta$ 为步进角），用泰勒展开近似：
$$\cos\Delta\theta \approx 1 - \frac{(\Delta\theta)^2}{2}, \quad \sin\Delta\theta \approx \Delta\theta - \frac{(\Delta\theta)^3}{6}$$

插补误差（余项）：$|R_2(\cos)| \leq \dfrac{(\Delta\theta)^4}{24}$

#### Stage 1（0–60帧）：题目展示
```
ExampleBox：
  标题：IM案例2 · 数控圆弧插补的泰勒近似
  加工参数：圆弧半径 R=100mm，步进角 Δθ=0.001rad（约0.057°）
  问题：(1) 用2阶泰勒展开计算 cosΔθ 和 sinΔθ；
        (2) 估算插补误差（弦长误差）
```

#### Stage 2（60–150帧）：审题分析
1. 🏭 **工程背景**：实时控制中避免调用三角函数（耗时），用泰勒多项式替代
2. 📐 **展开公式**：$\cos x=1-x^2/2+x^4/24-\cdots$，$\sin x = x-x^3/6+x^5/120-\cdots$
3. ⚖️ **误差控制**：保留几项由加工精度要求决定（1μm精度需几阶？）
4. 🎯 **实际意义**：步进角 0.001rad 时，哪项可以忽略？

#### Stage 3（150–330帧）：分步计算
| 步骤 | 帧区间 | 内容 |
|------|--------|------|
| Step 1 | 150–195 | $\cos(0.001) \approx 1-\dfrac{(0.001)^2}{2} = 1-5\times10^{-7} = 0.9999995$ |
| Step 2 | 195–240 | $\sin(0.001) \approx 0.001-\dfrac{(0.001)^3}{6} = 0.001-1.67\times10^{-10}\approx 0.001$ |
| Step 3 | 240–285 | 余弦误差：$|R_2|\leq\dfrac{(0.001)^4}{24}=4.17\times10^{-14}$（极小）|
| Step 4 | 285–330 | 弦长误差：$\Delta L = R\cdot\Delta\theta - R\sin\Delta\theta \approx R\cdot\dfrac{(\Delta\theta)^3}{6}=100\times\dfrac{10^{-9}}{6}\approx1.67\times10^{-8}$ mm $= 0.017$ nm（远小于1μm✓） |

#### Stage 4（330–390帧）：TheoremBox 结论
```
TheoremBox：
  标题：插补误差分析
  近似值：cos(0.001) ≈ 0.9999995，sin(0.001) ≈ 0.001
  弦长误差：≈ 0.017 nm（远小于加工精度 1μm）
  工程结论：Δθ=0.001rad 时，2阶泰勒展开精度完全满足数控加工要求
```

#### Stage 5（390–480帧）：3D 可视化
```
CoordinateSystem3D（rotateY: 0→360）：
  - ParametricCurve3D 绘制圆弧（R=100，一段约30°的圆弧）
  - 圆弧上标注若干插补点（均匀步进角）
  - 弦线（直线段）连接相邻插补点（半透明红色）
  - 弦长误差用放大镜局部放大（动画在圆弧上局部聚焦）
  - 正弦/余弦精确值与泰勒近似值对比曲线（右侧小图）
  - 旋转驱动：rotateY = interpolate(frame,[390,480],[0,360],clamp)
```

#### Stage 6（480–540帧）：总结
```
✅ 泰勒展开将三角函数计算简化为加减乘运算（嵌入式系统友好）
✅ 误差控制：余项估计给出精度保证，指导截断阶数选择
✅ 工程规律：步进角每减半，弦长误差减小约8倍（三次方关系）
```

### IM03：振动信号的傅里叶频谱分析

**文件名建议**：`Im03_VibrationSpectrum.tsx`
**Composition ID**：`Ch12-IM03-VibrationSpectrum`
**durationInFrames**：`540`

**工程背景**：
工业设备健康监测中，振动信号的傅里叶分析是识别故障特征频率的核心方法。齿轮磨损产生特定频率（啮合频率 = 转速×齿数），轴承故障产生包络频率。通过傅里叶级数/变换，将时域振动信号分解为频域分量，实现故障诊断。

**数学模型**：
振动信号（含基频和谐波）：
$$x(t) = A_1\sin(\omega_0 t) + A_2\sin(2\omega_0 t) + A_3\sin(3\omega_0 t) + n(t)$$

傅里叶系数揭示各频率成分的幅度：
$$A_k = \frac{2}{T}\int_0^T x(t)\sin(k\omega_0 t)\,dt$$

#### Stage 1（0–60帧）：题目展示
```
ExampleBox：
  标题：IM案例3 · 齿轮箱振动故障频谱分析
  信号模型：x(t) = sin(100πt) + 0.5sin(200πt) + 0.3sin(300πt)
  基频：50Hz（电机转速 3000 rpm），2倍频和3倍频为谐波
  故障特征：3倍频幅度突增 → 可能的齿面损伤
  问题：计算前3阶傅里叶系数，判断故障特征
```

#### Stage 2（60–150帧）：审题分析
1. 🏭 **工程背景**：齿轮每转产生啮合冲击，谐波包含齿面状态信息
2. 📐 **傅里叶分析**：将信号分解到各频率分量，提取幅度谱
3. ⚖️ **正交性**：$\int_0^T\sin(m\omega t)\sin(n\omega t)dt = 0$（$m\neq n$），保证各频率独立提取
4. 🎯 **故障判断**：正常齿轮谐波幅度快速衰减；故障时特定谐波异常增大

#### Stage 3（150–330帧）：分步计算
